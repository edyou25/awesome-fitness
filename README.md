## 📋 训练计划

- 💪 **力量训练** - 初级到高级的力量训练方案
- 🏃 **有氧训练** - HIIT、稳态有氧、循环训练
- 🤸 **柔韧性训练** - 拉伸、瑜伽、灵活性训练
- 🏆 **专项训练** - 跑步、骑行、游泳等专项计划

## 🚀 快速开始

1. **评估水平** - 选择适合的训练级别
2. **选择重点** - 确定训练目标
3. **执行计划** - 按计划坚持训练
4. **跟踪进步** - 记录训练效果
